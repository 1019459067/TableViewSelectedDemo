# TableViewSelectedDemo --- 自定义
根据系统的全选，替换全选、取消全选的图片

## 替换系统的全选功能，本Demo主要采用xib完成。数据来源于模拟刷新

- 1、支持单选，取消单选
- 2、支持多选，取消多选
- 3、支持删除功能
- 4、支持cell的高度自适应
